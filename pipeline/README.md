# GAIA TA2 Pipeline

This new pipeline is based on [KGTK](https://github.com/usc-isi-i2/kgtk).


## Docker

Docker build:

```
make docker-build
```

Docker run:

```
make docker-run
```

All configurations can be found in Makefile.


## Generate resource

`df_wd_fb.csv`: wikidata to freebase mapping.
